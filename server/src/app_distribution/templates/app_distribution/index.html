Come installare le app
iOS
  1. Tocca "Installa su iOS" dall'app desiderata
  2. Safari chiederà il permesso per installare l'app
  3. Tocca "Installa" per confermare
  4. L'app apparirà nella home screen
  5. Vai in Impostazioni > Generali > Gestione dispositivi
  6. Autorizza il certificato dello sviluppatore
Android
  1. Tocca "Scarica APK" dall'app desiderata
  2. Il file APK verrà scaricato sul dispositivo
  3. Apri il file APK scaricato
  4. Android potrebbe chiedere di autorizzare installazioni da fonti sconosciute
  5. Segui le istruzioni per completare l'installazione
